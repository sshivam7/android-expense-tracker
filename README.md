# Android Expense Tracker Application 
A simple expense manager that allows users to track their income and expenses. It
allows users to add, view, and delete new entries, as well as see a summary of their expenses. 

The application makes use of:
* Notifications to remind the user to track their expenses
* The Room Persistence Library to store and retrieve expense data
* Material Design themes and widgets to style app components
* Other Android Jetpack components such as RecyclerViews, Fragments, and ViewModels.
