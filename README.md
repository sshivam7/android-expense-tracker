# Android Expense Tracker Application 
A simple expense manager that allows users to track their income and expenses. This application 
allows users to add, view, and delete new entries, as well as see a summary of their expenses. 

## Installation
Clone and import into Android Studio
```bash
git clone https://github.com/ShivamSood17/android-expense-tracker
```
